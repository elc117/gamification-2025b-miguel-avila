## MUBIFication
Nome provisório

## Ideia Geral

A ideia proposta é criar uma plataforma de catalogação de filmes gameficada. A inspiração inicial foi o [Letterboxd](https://letterboxd.com/), porém com a inserção de elementos oriundos de *games* para incentivar o usuário a usar o aplicativo.
